Blockchainbasiertes Dokumenten Tracking System

cleske@extern.schoenhofer.de 2018

Dies ist eine Demo des Dokumententracking-Systems für Schönhofer GmbH, Siegburg.

Dieses System demonstriert die Verfolgung von sicherheitsrelevanten Dokumenten mit Hilfe der Hyperledger Fabric Blockchain.

Das System simuliert:

  • drei unabhängigen Dienststellen (abgebildet als Domänen: "Amt1", "Amt2", "Amt3")
  • jeweils drei Benutzern pro Dienststelle (user1, user2, user3) und einen Dienststellen-Administrator
  • zehn global verfügbare und indexierte Beispiel-Dokumente mit Sicherheitsstufen
  • Sicherheitsstufen reichen von 0 bis 3 (= höchste Sicherheitsstufe) und sind Benutzern und Dokumenten zugewiesen
  • Benutzer müssen zur Ausleihe mindestens die Sicherheitsstufe des Dokumentes aufweisen
